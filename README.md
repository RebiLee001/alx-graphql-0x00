# GraphQL Character Query Task

## Objective
Write GraphQL queries to fetch specific character information using the `character(id: ID!)` field from the Rick and Morty GraphQL API.

## Endpoint
https://rickandmortyapi.com/graphql

## Requirements
For each character ID (1, 2, 3, 4):

- Create a `.graphql` file containing the query
- Create a corresponding `.json` file containing the API response
- Fields included in the query:  
  - `id`
  - `name`
  - `status`
  - `species`
  - `type`
  - `gender`

## Files
- `character-id-1.graphql`
- `character-id-1-output.json`
- `character-id-2.graphql`
- `character-id-2-output.json`
- `character-id-3.graphql`
- `character-id-3-output.json`
- `character-id-4.graphql`
- `character-id-4-output.json`
