# GraphQL Episode Queries

This folder contains GraphQL queries that retrieve specific episodes from the Rick and Morty GraphQL API.

**Endpoint:**
https://rickandmortyapi.com/graphql

## Tasks Completed
- Queries to fetch episode details by ID (1, 2, 3, 4)
- Fields included:
  - `id`
  - `name`
  - `air_date`
  - `episode`
